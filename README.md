- Clone repo</br>

    ```
    git clone git@github.com:Nimebox/Nimebox-api.git

  ```

- Install depedencies
  ```
  npm i
  ```

  ```
  npm run dev
  ```

  ```
  npm run build && npm start
  ```
- Running linter
  ```
  npm run lint
  ```
- Running tests
  ```
  npm test
  ```
